# Yadi Liu — Portfolio

## Publish and manage the site

1. Create a GitHub repository and upload every file in this folder.
2. Import that repository in Netlify and publish it as a static site.
3. In Netlify, enable **Identity**, invite your own email address, then enable **Git Gateway**.
4. Visit `your-site-address/admin/` to log in and manage the Black & White and Colour photography galleries.

The CMS saves uploaded photos in `assets/uploads` and publishes changes automatically after you click **Publish**.
